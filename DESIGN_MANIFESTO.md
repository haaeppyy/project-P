# Design Manifesto — portfolio-super

Nothing is implemented because it is technically interesting.
Everything is implemented because it improves the visual experience.

If an effect cannot be noticed by a first-time visitor, remove it.
If an animation attracts attention to itself, remove it.

Whitespace is a feature.
Simplicity is intentional.

Every pixel must justify its existence.

The goal is not to impress.
The goal is to feel inevitable.
